ARGB

ARGB stands for Addressable Red Green Blue(RGB) LED. In ARGB you can
class uqtpy.actuators.argb.ARGB(port=None, pin=None, ob=True, pixels=8)

ARGB stands for Addressable Red Green Blue(RGB) LED. In ARGB you can choose the color of any LED of your choice. Set the specified digital input pin to the provided value

Usage : set_color( 2,red=255) # Sets 3rd LED to Red set_color( 2,green=255) # Sets 3rd LED to Green fill_color( ) # turns Off all LEDs fill_color( green=255 ) # turns all LEDs Green fill_color( 255,255,255 ) # turns all LEDs White set_color( 2,green=255, show=False) # Sets internal value of 3rd LED to green value.Explicitly call self.show()
